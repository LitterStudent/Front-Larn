# NPM(Node Package Manager )node包管理器

1.是Node.js默认的包管理器

2.**npm init --yes** 默认初始化

3.npm  i -S 保存于生产时的依赖

4.npm i -D 保存于开发时的依赖，后续项目打包时只打包dependencies内的依赖。
